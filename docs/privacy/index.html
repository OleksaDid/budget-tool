Privacy Policy

Last updated: June 17, 2026

Personal Budget is a self-hosted personal finance application operated by the user for personal use. The application helps the user import, categorize, review, and analyze their own bank account and financial transaction data.

Data Collected

The application may process the following data when the user explicitly connects an account or imports a file:

  • Bank account metadata, such as account name, account type, masked IBAN, and currency.
  • Account balances.
  • Bank transactions, including dates, amounts, currencies, descriptions, counterparties, and transaction status.
  • User-created categories, vendors, notes, import history, and application settings.

How Data Is Used

Data is used only to provide personal budgeting, transaction review, categorization, cashflow analysis, and net-worth features to the user. Bank access is used only to retrieve account, balance, and transaction information authorized by the user.

Storage

Financial data is stored in the user's own self-hosted application database. The application is not operated as a public hosted service and does not sell, rent, or monetize user financial data.

Sharing

Personal Budget does not sell user data and does not share financial data with advertisers or unrelated third parties. Account data may be exchanged with an authorized open banking provider only as required to connect the user's bank account and retrieve data requested by the user.

Security

The application is designed to store sensitive credentials encrypted at rest and to use read-only bank account access for account information. The user remains responsible for operating and securing their own self-hosted deployment.

Revocation

The user may revoke bank access through their bank or open banking provider. The user may also remove stored application data from their own deployment.

Contact

For questions about this personal application, contact the operator of the self-hosted deployment.
